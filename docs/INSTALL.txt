mingtak.securitymethod Installation
-----------------------------------

To install mingtak.securitymethod using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``mingtak.securitymethod`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        mingtak.securitymethod
       
* Re-run buildout, e.g. with:

    $ ./bin/buildout
